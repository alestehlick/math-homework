Quick Review: Trigonometric Functions

All angles in this assignment reduce to the five “anchor angles” \(0^\circ, 30^\circ, 45^\circ, 60^\circ, 90^\circ\) (or in radians: \(0, \frac{\pi}{6}, \frac{\pi}{4}, \frac{\pi}{3}, \frac{\pi}{2}\)).

Obtention of the reference angle

Step 1: Reduce the angle to a reference angle (an acute angle made with the \(x\)-axis). You can subtract or add full turns: \(360^\circ\) or \(2\pi\).

Step 2: Decide the quadrant and the sign:
Quadrant I: all \(+\)   •   Quadrant II: \(\sin +\)   •   Quadrant III: \(\tan +\)   •   Quadrant IV: \(\cos +\)

Step 3: Use the exact-value table below (then attach the correct sign). Remember: \(\tan\theta=\dfrac{\sin\theta}{\cos\theta}\), and tangent is undefined when \(\cos\theta=0\) (at \(90^\circ, 270^\circ\) or \(\frac{\pi}{2}, \frac{3\pi}{2}\)).

Exact values table (memorize / recognize)

Angle \(\sin\) \(\cos\) \(\tan\)
\(0^\circ\) / \(0\) \(0\) \(1\) \(0\)
\(30^\circ\) / \(\frac{\pi}{6}\) \(\frac{1}{2}\) \(\frac{\sqrt{3}}{2}\) \(\frac{\sqrt{3}}{3}\)
\(45^\circ\) / \(\frac{\pi}{4}\) \(\frac{\sqrt{2}}{2}\) \(\frac{\sqrt{2}}{2}\) \(1\)
\(60^\circ\) / \(\frac{\pi}{3}\) \(\frac{\sqrt{3}}{2}\) \(\frac{1}{2}\) \(\sqrt{3}\)
\(90^\circ\) / \(\frac{\pi}{2}\) \(1\) \(0\) undefined
Common radian “translations”
\(\frac{\pi}{6}=30^\circ\), \(\frac{\pi}{4}=45^\circ\), \(\frac{\pi}{3}=60^\circ\), \(\frac{\pi}{2}=90^\circ\). Also: \(2\pi=360^\circ\).
