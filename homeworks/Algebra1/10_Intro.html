Reading: Roots (Square, Cube, Fourth, …)

A root is a way to “undo” an exponent. For example, squaring takes a number \(x\) and produces \(x^2\). The square root goes the other way: it answers the question, “what number squared gives this?”

Start with simple examples (square roots)

Example 1
\(\sqrt{9}=3\) because \(3^2=9\).
Example 2
\(\sqrt{49}=7\) because \(7^2=49\).
Example 3
\(\sqrt{0}=0\) because \(0^2=0\).
But wait… what about negatives?
\((-3)^2=9\) is also true. So why don’t we say \(\sqrt{9}=-3\) too?

Here is the key idea: the symbol \(\sqrt{\phantom{a}}\) is meant to be a single answer. In math language, it is a function: one input → one output.

The equation \(x^2=9\) has two solutions (\(x=3\) and \(x=-3\)). But the symbol \(\sqrt{9}\) is defined to mean the positive one. That way, \(\sqrt{9}\) is not “two answers.” It is one clear answer.

Definition (square root)
For \(a\ge 0\), \(\sqrt{a}\) means the positive number \(x\) such that \(x^2=a\).

Cube roots (odd roots behave differently)

Example 4
\(\sqrt[3]{27}=3\) because \(3^3=27\).
Example 5
\(\sqrt[3]{-8}=-2\) because \((-2)^3=-8\).

Notice something important: with odd powers like cubes, a negative number stays negative. So the cube root of a negative number is a real number. (That is not true for square roots.)

Fourth roots, sixth roots, … (even roots)

Example 6
\(\sqrt[4]{16}=2\) because \(2^4=16\).
Example 7
\(\sqrt[6]{64}=2\) because \(2^6=64\).
Why do even roots choose the positive answer?
Because even powers erase the sign: \((2)^4=16\) and \((-2)^4=16\). So the equation \(x^4=16\) has two real solutions (\(x=\pm 2\)). But the symbol \(\sqrt[4]{16}\) is defined to be the positive one, so it gives one consistent answer.
Definition (n-th root)
If \(n\) is a positive integer, then \(\sqrt[n]{a}\) means a number \(x\) such that \(x^n=a\).

• If \(n\) is even, then \(\sqrt[n]{a}\) is defined only for \(a\ge 0\), and it means the positive solution.
• If \(n\) is odd, then \(\sqrt[n]{a}\) exists for any real \(a\), and there is exactly one real solution.

So remember: even root → needs \(a\ge 0\) and gives the positive answer.     odd root → works for negatives too.
